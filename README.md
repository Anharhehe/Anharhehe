## 🚀 Featured Projects

### 🥽 [SightWise — AI Smart Glasses for Visually Impaired](https://github.com/anharhehe) *(Final Year Project)*
`YOLOv8` · `TFLite INT8` · `Raspberry Pi 4B` · `Vosk STT` · `Tesseract OCR` · `OpenCV` · `GPS`

> Complete AI-powered smart glasses system for visually impaired individuals — real-time edge inference for indoor/outdoor object detection, **99.9% accuracy** Pakistani currency recognition, OCR for text reading, offline GPS navigation, and a fully voice-controlled interface. All AI modules run on Raspberry Pi 4B via INT8 quantized models within power constraints.

---

### 🩺 [BreastCare AI — Deep Learning Cancer Detection](https://github.com/anharhehe)
`Swin Transformer` · `FastAPI` · `React.js` · `PyTorch` · `CBIS-DDSM` · `HuggingFace`

> Full-stack AI web app for breast cancer detection using a fine-tuned Swin Transformer on 30,000+ mammography images — **87.5% accuracy, 0.884 ROC-AUC**. Dual-view ensemble (CC + MLO) mimics radiologist workflow. Includes drag-and-drop upload, real-time results, and automated PDF report generation.

---

### 🚗 [FogSight — Real-Time Vehicle Detection in Fog](https://github.com/anharhehe) *(In Progress)*
`YOLOv8` · `React Native` · `OpenCV` · `Image Dehazing` · `Ultrasonic Sensor` · `Hugging Face`

> AI-powered road safety system detecting vehicles in foggy conditions via mobile app. Image dehazing pipeline enhances visibility before YOLOv8 inference. Ultrasonic proximity sensor triggers real-time audio alerts — no vehicle modifications needed.

---

### 🧬 [Depression Detection from Visual Content](https://github.com/anharhehe)
`ResNet18` · `Logistic Regression` · `PyTorch` · `Scikit-learn` · `OpenCV`

> Binary classification system for detecting depression indicators from image data (20,186 images). Compared traditional ML (Logistic Regression — 75%) vs deep learning (ResNet18 transfer learning), analyzing precision/recall trade-offs for deployment recommendations.

---

### 💬 [Sarcasm Detection on Reddit](https://github.com/anharhehe)
`DistilBERT` · `TF-IDF` · `Logistic Regression` · `Naive Bayes` · `NLP`

> Comprehensive comparative study on 1.3 million Reddit comments. DistilBERT achieved **76.65% accuracy** vs TF-IDF + LR (70.80%) and Naive Bayes (66.55%). Evidence-based selection recommendations balancing performance, interpretability, and compute cost.

---

### 🤖 [Ergonomic Posture Assessment Agent](https://github.com/anharhehe)
